  <action>
Create `stream/mapping/generic_generator.py` with the `GenericMappingGenerator` class. The class:

**Constructor:** `__init__(self, accelerator: Accelerator, workload: Workload, output_dir: str)`
- Store accelerator, workload, output_dir

**Public method:** `generate_all_groups(self) -> list[str]`
- Call `self.workload.split_fusion_groups()` to get `list[Workload]` (sub-workloads)
- For each sub-workload (indexed `i`): call `self._generate_group_yaml(sub_workload, i)` to get the YAML file path
- Return list of all YAML paths

**Private method:** `_generate_group_yaml(self, sub_workload: Workload, group_idx: int) -> str`
- Build the mapping dict via `self._build_mapping_dict(sub_workload)`
- Validate via `MappingValidator(mapping_dict).validate()` — raise ValueError if invalid
- Write to `{self.output_dir}/group_{group_idx}/mapping.yaml` using `yaml.safe_dump`
- Return the file path

**Private method:** `_build_mapping_dict(self, sub_workload: Workload) -> dict`
- Get computation nodes: `cns = sub_workload.get_computation_nodes()`
- For each node, determine its core allocation via `_select_cores_for_node(node)`
- Build `layers` list entries with:
  - `name`: `cn.name`
  - `core_allocation`: `[[c.id for c in cores]]` (nested list, one slot)
  - `inter_core_tiling`: `[[{"dim": f"D{dim_idx}", "split": len(cores)}]]` if len(cores) > 1, else `[]`
  - The split dimension: find the largest node-local dimension whose size is divisible by len(cores). Use `sub_workload.get_dims(cn)` to get dims, `sub_workload.get_dimension_size(dim)` for sizes. Iterate from largest to smallest, pick first divisible by n_cores. If none divisible, use largest dim with split = n_cores (relaxed divisibility — the pipeline will handle this via determine_fusion_splits).
- Build `fused_groups` list with single entry:
  - `name`: `"Fused_Group_1"`
  - `layers`: `[cn.name for cn in cns]` (only ComputationNode names)
  - `intra_core_tiling`: result of `_build_intra_core_tiling(sub_workload, cns)`
- Return `{"layers": layers, "fused_groups": [fused_group]}`

**Private method:** `_select_cores_for_node(self, node: ComputationNode) -> list[Core]`
- Get node op type: `node.type` (e.g., "Conv", "MaxPool", "Add")
- Filter `self.accelerator.core_list` by:
  - Skip cores with `core.type in ("offchip", "shim")`
  - If `core.operator_types is None`: core accepts all ops, include it
  - If `core.operator_types is not None` and `node.type in core.operator_types`: include it
  - If `core.operator_types is not None` and `node.type not in core.operator_types`: exclude it
- If result is empty: fall back to all cores with `core.type == "compute"` (per D-06 fallback)
- Per D-09: If only one core matches (specialized pooling/simd), return just that core
- Per D-10: If multiple compute cores match, return all of them (full parallelism)

**Private method:** `_find_split_dim(self, sub_workload: Workload, cn: ComputationNode, n_cores: int) -> int | None`
- Get node dims: `dims = sub_workload.get_dims(cn)`
- For each dim (sorted by size descending):
  - `size = sub_workload.get_dimension_size(dim)`
  - If `size >= n_cores` and `size % n_cores == 0`: return `dims.index(dim)` (the D-index)
- If no perfect divisor found, use the largest dim index anyway (will be n_cores split)
- Return the dim index (int) or None if node has no dims

**Private method:** `_build_intra_core_tiling(self, sub_workload: Workload, cns: tuple[ComputationNode, ...]) -> list[dict]`
- Per D-08: memory-aware tiling. Use first ComputationNode as reference for dim naming.
- Pick the first CN that has dimensions: `ref_cn = cns[0]`
- Get its dims: `dims = sub_workload.get_dims(ref_cn)`
- If no dims, return `[]`
- Pick the first dim (index 0) as the intra-core tiling dimension
- Compute tile size: `dim_size = sub_workload.get_dimension_size(dims[0])`
  - Tile = `dim_size` (full dimension fits — TPU SRAM is 2MB, tensors are small)
  - Actually per open question 3 in RESEARCH.md: use `tile_size = dim_size // max(1, n_cores)` where n_cores is from the ref node's allocation (ensures divisibility with inter_core split). But simplest valid approach: `tile_size = dim_size` (no temporal loop for intra_core — valid, produces nb_splits=1 in determine_fusion_splits).
  - To be safe and ensure divisibility: `tile_size = dim_size` (when no inter_core split on this dim) or use a divisor that satisfies `dim_size % (tile_size * spatial_unrolling) == 0`.
  - Simplest correct implementation: `tile_size = dim_size` — this means no temporal splitting, which is always valid.
- Return `[{"dim": f"{ref_cn.name}.D0", "tile": tile_size}]`
