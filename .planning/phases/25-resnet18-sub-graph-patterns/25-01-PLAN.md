```python
from enum import Enum, auto
from dataclasses import dataclass

class ResNetPattern(Enum):
    BASIC_RESIDUAL = auto()       # Conv->Relu->Conv->Add with skip
    STRIDE2_DOWNSAMPLE = auto()   # Conv(s=2)->Relu->Conv + Conv(1x1,s=2)->Add
    FRONTEND = auto()             # Conv(7x7,s=2)->Relu->MaxPool(s=2)
    DUAL_RESIDUAL = auto()        # Two residual blocks + Reshape FusionEdge between

@dataclass
class ResNetSubgraphConfig:
    pattern: ResNetPattern
    batch_size: int = 1
    in_channels: int = 8
    height: int = 16
    width: int = 16
    out_channels: int = 8       # same as in_channels for basic residual (skip dim must match)
    in_dtype: str = "bf16"
    weight_dtype: str = "bf16"
```

```python
def make_resnet_subgraph(config: ResNetSubgraphConfig | None = None) -> str:
```

**BASIC_RESIDUAL graph:**

**STRIDE2_DOWNSAMPLE graph:**
```
input: [1, C_in, H, W]   (C_in=8, H=32, W=32 recommended for stride-2)
Main path:
  Conv1(input, w1, kernel=3, pad=1, stride=2) -> conv1_out    [1, C_out, H/2, W/2]
  Relu1(conv1_out) -> relu1_out                                 [1, C_out, H/2, W/2]
  Conv2(relu1_out, w2, kernel=3, pad=1, stride=1) -> conv2_out [1, C_out, H/2, W/2]
Downsample path:
  Conv3(input, w3, kernel=1, pad=0, stride=2) -> ds_out        [1, C_out, H/2, W/2]
Add1(conv2_out, ds_out) -> add1_out                            [1, C_out, H/2, W/2]
output: add1_out
```
- Use `config.height=32, config.width=32` for this pattern (override defaults)
- `out_channels` = 16 (doubles from in_channels=8)
- Weight shapes: w1=[16,8,3,3], w2=[16,16,3,3], w3=[16,8,1,1]
- Spatial math: (32+2*1-3)//2+1 = 16 for main; (32+0-1)//2+1 = 16 for downsample. Match!
- Override config internally: if pattern is STRIDE2_DOWNSAMPLE, use height=32, width=32, out_channels=16

**FRONTEND graph:**

```
input: [1, 3, H, W]   (3 input channels like RGB; H=32, W=32)
Conv1(input, w1, kernel=7, pad=3, stride=2) -> conv1_out  [1, C_out, H/2, W/2]
Relu1(conv1_out) -> relu1_out                              [1, C_out, H/2, W/2]
MaxPool1(relu1_out, kernel=3, stride=2, pad=1) -> pool_out [1, C_out, H/4, W/4]
output: pool_out
```
- Use `in_channels=3` (RGB), `height=32, width=32`, `out_channels=8`
- Conv spatial: (32+6-7)//2+1 = 16. MaxPool spatial: (16+2-3)//2+1 = 8. Correct.
- Override config internally for this pattern: in_channels=3, height=32, width=32, out_channels=8

**DUAL_RESIDUAL graph:**

```
input: [1, C, H, W]
Block 1:
  Conv1(input, w1, kernel=3, pad=1) -> conv1_out           [1, C, H, W]
  Relu1(conv1_out) -> relu1_out                             [1, C, H, W]

FusionEdge boundary:

  Conv4(relu2_out, w4, kernel=3, pad=1) -> conv4_out       [1, C, H, W]

output: add2_out
```

- Block 2 skip connection: Add2 takes conv4_out and reshape_out (fan-out from reshape_out InEdge in sub-workload 2)
